## Decisions

- **entryNames disabled for build-clean**: the package has `src/types.ts` and `src/core/types.ts` which collide when flattened to `[name]`.
- **packages/watch removed from tsup config**: `exactOptionalPropertyTypes: true` in this package's tsconfig makes `watch: config.watch` a type error (undefined not assignable to optional property). `packages: 'external'` not in tsup's Options type.
- **Biome config split**: root biome.json owns formatter/assist, per-package biome.json owns linter rules. Uses `"extends": "//"` with `"root": false` so packages inherit formatting from root but define their own lint rules.
- **minify/treeshake both false**: decided with SC during session. terser devDependency removed as a result.
